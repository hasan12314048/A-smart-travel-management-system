# Smart Travel Management System (C)

A console-based travel management system developed in C as a first-semester university project.

## Features
- User Sign Up & Login (File Handling)
- Air Ticket Booking
- Bus Ticket Booking
- Train Ticket Booking
- Student Transport System
- Menu-driven interface

## Concepts Used
- C Programming
- File Handling
- Structures
- Functions
- Conditional Logic
- Menu-based UI

## Limitations
- Single-user support
- Plain-text password storage
- Platform dependent (Windows)
